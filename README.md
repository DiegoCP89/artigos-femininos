## 🛠️ Gestão e Metodologia
Para o gerenciamento deste projeto, utilizamos o **GitHub Projects** integrado com metodologia **Kanban**. Cada artefato é rastreado como uma *Issue*, categorizada por Fases do Ciclo de Vida de Software (SDLC), por meio de Labels específicas e organizada pelos marcos de entrega (Milestones: Planejamento, Elicitação de Requisitos e Projeto).
