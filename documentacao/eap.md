# 📂 EAP - Estrutura Analítica do Projeto

> Este artefato apresenta a decomposição hierárquica do escopo do projeto, focada nas entregas do sistema de inventário digital.

---

### 🌳 Hierarquia de Entregas

1. **Gestão do Projeto**
   * 1.1 Iniciação (TAP, SWOT, 5W2H)
   * 1.2 Planejamento e EAP
   * 1.3 Monitoramento e Controle

2. **Requisitos e Análise**
   * 2.1 Elicitação de Requisitos (Entrevistas/Questionários)
   * 2.2 Documentação de Requisitos e Matriz de Rastreabilidade

3. **Design e Arquitetura**
   * 3.1 Modelagem de Dados (MER/DER)
   * 3.2 Diagramas UML (Classe, Caso de Uso, Atividades)
   * 3.3 Prototipação de Telas (UI/UX)

4. **Desenvolvimento (C#)**
   * 4.1 Implementação do Banco de Dados (PostgreSQL/SQL Server)
   * 4.2 Desenvolvimento do Módulo de Cadastro
   * 4.3 Desenvolvimento do Módulo de Movimentação de Estoque
   * 4.4 Implementação de Relatórios e Alertas

5. **Encerramento**
   * 5.1 Testes de Software e QA
   * 5.2 Treinamento de Usuários
   * 5.3 Entrega Final e Manual do Sistema
