# 🎯 Identidade Organizacional

> Este documento define a base estratégica do projeto para a disciplina de Engenharia de Software III.

---

## 🚀 Missão
**Fornecer acessórios para realçar a beleza feminina.**
&nbsp;

## 👁️ Visão
**Ser reconhecida como a loja amiga da mulher até 2032.**
&nbsp;

## 💎 Valores
* **Liberdade:** Incentivar cada cliente a expressar a sua beleza.
* **Respeito:** Tratar todas as pessoas com consideração e respeito. 
* **Dedicação:** Empenho em oferecer sempre o melhor.
* **Excelência:** Superar expectativas em cada experiência.
